MOTIVE TRIGGER

this is a c# project that allows to send triggers to Motive 
and so to start and stop the recording of takes.

The code allows also to set the name of the track to be recorded


Motive settings required:
- Data Streaming:
Broadcast Frame: ON
Local Interface: loopback
Asset Markers: ON
Rigid Bodies: ON
Remote Trigger: ON
Transmission Type: Multicast
Command Port: 1510
Multicast as Broadcast: ON
